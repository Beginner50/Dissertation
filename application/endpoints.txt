The bearer auth token will need to store:
	- UserID
	- Role
	- ProjectIDs
ProjectIDs to account for supervisors having multiple projects
New access token will need to be resent at project creation


Meeting Endpoints:
Authorization checks are performed at the controller/services level rather than at the middleware level.

Get supervisor's meetings:
	GET /meetings

	Standard auth check

Book a meeting:
	POST /projects/1/meetings
	
	ProjectID from params compared with that in access token

Edit a meeting's description:
	PATCH /projects/1/meetings/1

	ProjectID from bearer auth will be compared against that of the meeting before edit
	ProjectID is unique to a supervisor-student pair and therefore only allows them to edit.

Cancel a meeting:
	DELETE /organizers/1/meetings/1

	UserID (token) compared against userID (params)

Accept a meeting:
	POST /attendees/1/meetings/1/accept

	UserID (token) compared against userID (params)

Reject a meeting:
	POST /attendees/1/meetings/1/reject

	UserID (token) compared against userID (params)

Postpone a meeting:
	POST /attendees/1/meetings/1/postpone

	UserID (token) compared against userID (params)


Project:
Get Projects
	GET /projects

	Only allowed for GetProjects Permission (Supervisors & Admins)

Get Project
	GET /projects/1

	ProjectID from bearer auth will be compared with url params

Create a Project:
	POST /users/1/projects
	
	UserID (token) checked against userID (params)
	Database constraints ensure that Students can conduct only 1 project
	
Edit a Project:
	PATCH /users/1/projects/1

	Edit a project's title or description
	Allowed for both Students & Supervisors
	
Join a Project:
	POST /supervisors/1/projects/1/join

	UserID (token) checked against supervisorID (params) &
	role (token) checked against supervisor

Assign a Student to Project:
	POST /supervisors/1/projects/1/assign/1

	UserID (token) checked against supervisorID (params) &
	ProjectID (token) checked against projectID (params) & 
	role (token) checked against supervisor

Archive a Project
	DELETE /supervisors/1/projects/1

	Archive a project
